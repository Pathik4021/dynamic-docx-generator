# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-09

### Added
- Initial release
- `DocxGenerator` class with fluent API
- Template loading from file path or Buffer
- Placeholder replacement in document body, header, and footer
- Dynamic table generation with customizable styling
- Image insertion from file, URL, or Buffer
- Support for EMU (English Metric Units) for image dimensions
- TypeScript type definitions
- Utility functions for string manipulation, XML processing, and image handling
